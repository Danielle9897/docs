# Configuration: Update Options

{PANEL:Updates.Channel}

Indicates what release channel should be used to perform latest version checks.

- **Type**: `string`
- **Default**: `Patch`
- **Scope**: Server-wide only

Possible values: 

- `Stable`
- `Patch`
- `Dev`

{PANEL/}

{PANEL: Updates.BackgroundChecks.Disable}

Disable background latest version checks.

- **Type**: `boolean`
- **Default**: `false`
- **Scope**: Server-wide only

{PANEL/}
